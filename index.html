Prompt Variation Study

This project analyzes how subtle changes in wording influence AI-generated image outputs using Google Nano Banana 2. Three prompts were tested using the same core subject while adjusting descriptive intensity and specificity. The objective was to observe shifts in composition, lighting interpretation, stylistic rendering, and environmental detail.

Prompt 1 – Basic Description

The first prompt establishes a neutral baseline. It contains minimal modifiers and focuses only on subject and setting. The resulting image demonstrates standard composition and limited stylistic emphasis. This version serves as the control comparison for later outputs.

Prompt 2 – Detailed Enhancement

The second prompt introduces additional lighting descriptors and stylistic modifiers. Compared to the baseline, the output displays stronger tonal contrast, increased texture detail, and more deliberate environmental depth. This demonstrates the model’s sensitivity to descriptive layering.

Prompt 3 – Specific Cinematic Framing

The third prompt applies highly specific atmospheric and cinematic language. The generated result exhibits enhanced mood, directional lighting, and more intentional framing. This confirms that specificity significantly influences output structure and visual storytelling.

Comparative Insight

Across all three tests, Google Nano Banana 2 demonstrated clear responsiveness to prompt specificity. Increased descriptive precision directly correlated with more refined lighting behavior, environmental cohesion, and compositional intention. These results highlight the importance of structured prompt engineering when seeking predictable and stylistically controlled outputs.
