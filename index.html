Prompt Engineering & Output Sensitivity

Evaluating Visual Variability in Google Nano Banana 2

Research Objective

This study investigates how incremental prompt modification affects generative image output using Google Nano Banana 2. Three controlled prompts were constructed using the same base subject while systematically increasing descriptive specificity and environmental direction. The goal was to measure changes in compositional framing, lighting interpretation, atmospheric cohesion, and stylistic fidelity.

Prompt 1 – Baseline

The baseline prompt contains minimal descriptive modifiers. Output demonstrates neutral framing, balanced lighting, and limited stylistic bias. This image serves as the control condition for comparative evaluation.

Prompt 2 – Descriptive Expansion

The second prompt introduces additional environmental and lighting descriptors. The output reflects increased tonal contrast, stronger depth separation, and more defined spatial structure.

Prompt 3 – Cinematic Specificity

The final prompt incorporates cinematic language and atmospheric emphasis. Results show directional lighting, mood amplification, and tighter compositional control, indicating high responsiveness to linguistic precision.

Comparative Findings

Analysis confirms a direct relationship between prompt specificity and output refinement. Increased descriptive precision yielded stronger environmental consistency, clearer subject isolation, and enhanced narrative structure. These findings reinforce the importance of structured prompt engineering when seeking predictable generative outcomes.
