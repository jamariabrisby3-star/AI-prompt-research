AI Prompt Variation Study

This research evaluates how minor wording adjustments influence generated image outputs using Google Nano Banana 2. Three structured prompt variations were tested to measure stylistic consistency, compositional accuracy, and atmospheric interpretation.

Three-Prompt Test Results

Prompt 1 establishes a baseline description with minimal modifiers. This version serves as the control output.

Prompt 2 introduces stylistic and environmental descriptors, resulting in noticeable tonal and compositional shifts.

Prompt 3 applies specific atmospheric and cinematic language, demonstrating increased responsiveness to contextual modifiers.
